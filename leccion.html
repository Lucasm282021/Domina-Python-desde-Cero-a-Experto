Fundamentos: Variables y Tipos

Python es un lenguaje de programación con tipado dinámico. No necesitas declarar el tipo de una variable al momento de crearla.

Tipos Nativos

  • Enteros (int): edad = 25
  • Flotantes (float): pi = 3.1415
  • Cadenas (str): nombre = "Ana"
  • Booleanos (bool): es_mayor = True

Consejo: Usa type(variable) para conocer de qué tipo es ese dato bajo el capó.
